## Instalación en Linux (Ubuntu)
```bash
sudo apt update
sudo apt install nodejs npm
```


## Instalación en MacOS
```bash
brew install node
```


## Verificar la instalación
```bash
node -v
```

```
v20.9.0
```

```bash
npm -v
```
